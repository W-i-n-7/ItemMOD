2. Installing/updating ItemMOD (Crustacean)

After you have ran Crustacean and installed BepInEx, You should be able to use this button to install ItemMOD (if this doesnt work then see manual method below)



3. Installing/updating ItemMOD (Manual)

1. Download the .dll file 2: Open Steam 3: Navigate to Library 4: Find "Crab Game" 5: Right-click and Select Properties 6: In the "Properties" window, go to the "Local Files" tab. 7: Click "Browse Local Files" 8. Drag itemMOD.dll to your BepInEx\plugins folder
